Generate a short ELN (electronic lab notebook) entry summarizing what we just did
or found in this session, formatted for direct copy-paste into Signals Revvity.

Structure it as:

**Date:** [today's date]
**Task:** [one line — what was being worked on]

**What was done:**
- [2-4 bullet points, plain language, past tense, no code unless a specific
  command/parameter matters for reproducibility]

**Finding / Result:**
- [1-3 bullets — the actual outcome. If numeric/data-based, include the key
  numbers. If this is a troubleshooting note, state what the problem was and
  what fixed it, or what's still unresolved.]

**Next step:**
- [1 line — what happens next, if applicable]

Rules:
- Keep it short — this is a notebook entry, not a report. Aim for under 150 words.
- Write in plain declarative sentences, not marketing/summary language.
- Do not include code blocks or file paths unless they are essential for
  someone to reproduce the exact step later.
- If something failed or is still uncertain, say so plainly rather than
  smoothing it over — this is a record, not a pitch.
- Do not invent numbers, results, or outcomes not actually produced in this
  session.
